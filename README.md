# JH AirTracker

JH AirTracker brings Jackson Hole airport tools, flight information, personal notes, contacts, travel shortcuts, and app settings into one clean Android experience.

It is built for people who want quick access to airport-focused tools without jumping between pages, bookmarks, notes apps, and settings screens.

## What You Can Do

| Feature | What it helps with |
| --- | --- |
| Home | Open airport tools, live cameras, travel shortcuts, QR tools, notes, contacts, and profile options from one place. |
| Flights | Check arrivals, departures, flight status, tracking links, and airport-focused flight details. |
| Notes | Write notes, add titles, attach photos or files, record voice notes, set reminders, search, sort, and customize the layout. |
| Sync Online | Keep notes available through your account when online sync is turned on. |
| Contacts | Save and manage useful contacts for travel, airport work, or personal use. |
| Profile | Sign in, manage account details, and access security and privacy options. |
| Settings | Change app appearance, app icons, language, Liquid Glass style, feedback, privacy, and software updates. |

## Notes

Notes can stay local on your device or sync online when you sign in and turn on Sync Online.

You can use notes for quick reminders, travel details, airport tasks, checklists, voice notes, photos, files, and anything you want to keep close while using the app.

## Liquid Glass

JH AirTracker includes a Liquid Glass style that can be adjusted from clear to tinted. The app lets you tune the look so tabs, menus, and sheets feel easier to see in light or dark mode.

## Software Updates

The app can check for new versions and show what changed before you update. Release notes are shown inside the Software Update screen, so this README does not need a long changelog that changes every release.

## Built For

- Jackson Hole airport information
- quick flight checks
- travel and airport shortcuts
- personal notes with attachments
- clean light and dark mode use
- people who want everything in one app instead of scattered across tools
