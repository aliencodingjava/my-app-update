JH AirTracker brings Jackson Hole airport tools, flight information, personal notes, contacts, travel shortcuts, and app settings into one clean Android experience.

It is made for quick checks, daily airport use, travel planning, and keeping useful information close without jumping between separate apps.

## What You Can Do

| Feature | What it helps with |
| --- | --- |
| Home | Open airport tools, live cameras, travel shortcuts, QR tools, notes, contacts, and profile options from one place. |
| Flights | Check arrivals, departures, flight status, tracking links, and airport-focused flight details. |
| Notes | Write notes, add smart titles, attach photos and files, record voice notes, set reminders, search, sort, and choose the layout you like. |
| Sync Online | Keep notes connected to your account when Sync Online is turned on. |
| Contacts | Save useful contacts for travel, airport work, or personal use. |
| Profile | Sign in, manage account details, and keep account features available across sessions. |
| Settings | Adjust app appearance, language, app icons, Liquid Glass style, privacy, feedback, and software updates. |

## Notes

Notes can stay local on your device, or they can sync online when you sign in and turn on Sync Online.

You can use notes for travel details, airport tasks, checklists, voice reminders, photos, files, and anything else you want nearby while using the app. When online sync is active, notes can restore with their saved details so your account feels connected instead of starting over.

## Liquid Glass

JH AirTracker includes an adjustable Liquid Glass style. You can move between a clearer look and a more tinted look, then use Adaptive Luminance when you want glass surfaces to react better to the content behind them.

Tabs, menus, sheets, and note tools are designed to feel consistent in light and dark mode.

## Software Updates

The app can check for new versions, show what changed, and install updates directly from the app. Update notes are shown inside the Software Update screen, so this README stays focused on what the app does instead of becoming a long changelog.
